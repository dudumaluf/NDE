Como exportar do Mixamo (FBX → GLB)
  1. No Mixamo, escolha o personagem e baixe uma vez com Format: FBX Binary, Skin: With Skin — este é o 1º arquivo.
  2. Cada animação extra: baixe Without Skin, 30 fps, sem keyframe reduction (loops do Mixamo já fecham).
  3. Converta FBX → GLB no Blender: File → Import → FBX, depois File → Export → glTF 2.0 (.glb) — sem compressão Draco/Meshopt. (CLI no tools/README.md.)
  4. Arraste os GLB aqui em cima (personagem primeiro).
